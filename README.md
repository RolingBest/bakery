# Навчальний фронтенд-проєкт «Bakery»

Навчальний проєкт для практики **верстки** та **структурування стилів** без
використання фреймворків і сторонніх бібліотек.

## Мета проєкту

- **Відпрацювати семантичну HTML5-розмітку**
- **Поглибити розуміння CSS3** (позиціонування, сітки, адаптив, типографіка
  тощо)
- **Навчитися організовувати стилі на SCSS** з використанням методології **BEM**
- Відпрацювати структурування файлів і базову архітектуру стилів у реальному
  макеті

## Використані технології

- **HTML5** — семантична розмітка сторінок
- **CSS3** — базові стилі, адаптивність, анімації (за наявності)
- **SCSS (Sass)** — препроцесор для модульної організації стилів
- **BEM (Block-Element-Modifier)** — методологія іменування класів та
  структурування стилів
- **Без фреймворків** — чистий HTML/CSS/SCSS, без JS-фреймворків і UI-бібліотек
